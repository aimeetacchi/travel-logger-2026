# Travel Logger

A personal travel log to record every place you visit. Browse your trips by date range, and keep notes on the countries and cities you've been to.

Built with Next.js, Clerk authentication, Neon (PostgreSQL), and Drizzle ORM.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Commands

```bash
npm run dev      # start dev server at http://localhost:3000
npm run build    # production build
npm run lint     # run ESLint
```
